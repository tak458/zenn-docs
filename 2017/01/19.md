# Raspberry PI 3 model B を買った

## Raspbianをダウンロード

[raspbian jessie](http://ftp.jaist.ac.jp/pub/raspberrypi/raspbian/images/raspbian-2017-01-10/)

## パーツ買いたい

* ラズベリーパイ　ブレッドボード接続キット
* ブレッドボード
* ジャンパーコード
* 赤外線LED
* 通常LED
* 赤外線受信モジュール
* 抵抗

## 初期設定

* IP Address : 192.168.17.5/24
* Password : apple id と一緒
