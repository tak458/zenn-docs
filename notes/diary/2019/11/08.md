SSHでSynology NASに入って以下コマンドでJenkins起動
```
docker run \
  -v /var/run/docker.sock:/var/run/docker.sock \
  -v /volume1/docker/jenkins/home:/var/jenkins_home \
  --env-file /volume1/docker/jenkins/.env \
  -p 18085:50000 \
  -p 18080:8080 \
  --restart always \
  --user root \
  --name jenkins \
  -d jenkinsci/blueocean:latest
```
